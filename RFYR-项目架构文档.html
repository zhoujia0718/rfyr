📝 模块一：内容管理系统 核心模块

内容管理系统是整个平台的核心，负责管理所有类型的数字化内容，包括文章、笔记、大师观点、评论等。系统提供了完整的 CRUD 操作、富文本编辑能力和灵活的访问控制。

👑 模块三：会员订阅系统 核心模块

会员系统管理用户的订阅状态和会员权益。系统支持四个会员等级，每个等级对应不同的内容访问权限和功能限制。

3.1 会员等级体系

等级 名称 每日阅读 阅读时长 复盘访问 价格
FREE 免费用户 3篇 0.5小时 ❌ 免费
MONTHLY 月度会员 20篇 4小时 ❌ ¥XX/月
YEARLY 年度会员 50篇 8小时 ✅ ¥XX/年
PERMANENT 永久会员 无限 无限 ✅ 一次性

3.2 会员权益检查

lib/membership.ts
// 检查用户是否有权访问特定内容
async function canAccessArticle(userId: string, articleAccessLevel: string) {
  const membership = await getMembershipStatus(userId);
  
  if (!membership) return articleAccessLevel === 'free';
  
  const levelHierarchy = ['free', 'monthly', 'yearly', 'permanent'];
  const userLevel = levelHierarchy.indexOf(membership.tier);
  const articleLevel = levelHierarchy.indexOf(articleAccessLevel);
  
  return userLevel >= articleLevel;
}

// 白名单验证
const ALLOWED_ACTIONS = ['renew', 'cancel', 'upgrade', 'downgrade'];
const ALLOWED_PLANS = ['monthly', 'yearly', 'permanent'];

export async function POST(request: Request) {
  const body = await request.json();
  
  // 严格验证参数
  if (!ALLOWED_ACTIONS.includes(body.action)) {
    return NextResponse.json({ error: 'Invalid action' }, { status: 400 });
  }
  
  if (!ALLOWED_PLANS.includes(body.plan)) {
    return NextResponse.json({ error: 'Invalid plan' }, { status: 400 });
  }
}

function verifyAdminCookie(cookie: string): boolean {

  const expected = hmacSha256(SECRET, payload);
  return signature === expected;
}
